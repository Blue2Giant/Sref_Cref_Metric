- CSD ：原始是余弦相似度，范围 [-1, 1] ；现在在可视化里归一化为 [0, 1] 。
- DINOv2 ：余弦相似度，范围 [-1, 1] 。现在在可视化里归一化为 [0, 1] 。
- OneIG ：余弦相似度，范围 [-1, 1] 。现在在可视化里归一化为 [0, 1] 。
- CLIPCap（image-text） ：余弦相似度，范围 [-1, 1] 。现在在可视化里归一化为 [0, 1] 。
- CAS ：均方误差（MSE），范围 [0, +∞) 。 
上面都是越低越好

- OneAlign ：模型输出分数， 无强制边界 （取决于模型实现）。
- LAION aesthetic ：线性头输出， 无强制边界 （通常在 0–10 左右）。
- Aesthetic v2.5 ：logits 输出， 无强制边界 （通常在 0–10 左右）。
