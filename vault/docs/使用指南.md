## 快速开始

### 1. 安装和初始化

```python
import vault
from vault.storage.lanceduck.multimodal import MultiModalStorager

# 初始化存储
vault_path = "/path/to/your/vault"
MultiModalStorager.init(vault_path)
storager = MultiModalStorager(vault_path)
```

### 2. 创建和存储数据

```python
from vault.schema.multimodal import Image, Text, PackSequence

# 创建图像
image = Image.create(
    image_bytes,  # 图像字节数据
    uri="/path/to/image.jpg",
    source="my_dataset"
)

# 创建文本
text = Text.create(
    content="这是一段文本描述",
    uri="/path/to/text.txt",
    source="my_dataset"
)

# 创建序列
sequence = PackSequence.from_text_to_image(
    caption=text,
    image=image,
    source="my_dataset",
    uri="/path/to/sequence"
)

# 存储数据
storager.add_sequences([sequence])
storager.commit()
```

## 核心概念

### 数据层次结构

```
Vault (存储根目录)
├── PackSequence (序列) - 包含多个图像和文本的组合
│   ├── Image (图像) - 单个图像及其元数据
│   ├── Text (文本) - 单个文本及其元数据
│   └── Annotation (注释) - 额外的元数据信息
```

### 存储架构

- **Lance**: 存储大型二进制数据 (图像、文本内容)
- **DuckDB**: 存储结构化元数据和关系信息

## 数据模型

### Image (图像)

```python
@dataclass(frozen=True)
class Image:
    id: ID                    # 唯一标识符
    uri: str                  # 资源路径
    source: str               # 数据源
    pil_image: PIL.Image.Image # PIL 图像对象
    blob: bytes               # 图像字节数据
    annotations: list[Annotation] | None  # 注释信息
```

### Text (文本)

```python
@dataclass(frozen=True)
class Text:
    id: ID                    # 唯一标识符
    content: str              # 文本内容
    uri: str | None           # 资源路径
    source: str | None        # 数据源
    language: str | None      # 语言
    annotations: list[Annotation] | None  # 注释信息
```

### PackSequence (序列)

```python
@dataclass(frozen=True)
class PackSequence:
    id: ID                    # 唯一标识符
    images: Sequence[tuple[Image, str | int]]  # 图像及其索引
    texts: Sequence[tuple[Text, str | int]]    # 文本及其索引
    source: str               # 数据源
    uri: str                  # 资源路径
    annotations: list[Annotation] | None  # 注释信息
    meta: Any | None          # 自定义元数据
```

### Annotation (注释)

```python
@dataclass(frozen=True)
class Annotation:
    id: ID                    # 唯一标识符
    name: str                 # 注释名称
    type_: str | None         # 注释类型
    blob: bytes | None        # 二进制数据
    creator: Creator | None   # 创建者信息
    meta: Any | None          # 自定义元数据
```

results = storager.meta_hanlder.query_batch("""

## 自定义数据导入

### 创建自定义 Ingestor

```python
from vault.utils.ingest import BaseIngestor
from vault.schema.multimodal import PackSequence, Image, Text

@dataclass
class MyCustomIngestor(BaseIngestor):
    name: str = "MyCustomDataset"
    vault_path: str = "/path/to/vault"
    task_chunk_size: int = 100
    num_workers: int = 4
    
    # 数据源配置
    data_root: str = "/path/to/data"
    
    def prepare_tasks(self) -> list[Any]:
        """准备所有需要处理的任务"""
        # 返回任务列表，例如文件路径列表
        import os
        return [os.path.join(self.data_root, f) for f in os.listdir(self.data_root)]
    
    def process_tasks(self, tasks_chunk: list[Any], worker_id: int):
        """处理一批任务，生成 PackSequence 对象"""
        for task in tasks_chunk:
            # 处理单个任务
            image = Image.create(
                image_bytes=load_image(task),
                uri=task,
                source=self.name
            )
            
            text = Text.create(
                content=load_caption(task),
                uri=task + ".txt",
                source=self.name
            )
            
            sequence = PackSequence.from_text_to_image(
                caption=text,
                image=image,
                source=self.name,
                uri=task
            )
            
            yield sequence

# 使用自定义 Ingestor
ingestor = MyCustomIngestor()
ingestor.run()
```

### 处理不同数据格式

```python
def process_webdataset_tar(tar_path: str):
    """处理 WebDataset 格式的 tar 文件"""
    import webdataset as wds
    
    dataset = wds.DataPipeline(
        wds.SimpleShardList(tar_path),
        wds.atarfile_to_samples(),
        wds.rename(image="jpg;png;jpeg", text="txt")
    )
    
    for sample in dataset:
        image = Image.create(
            sample["image"],
            uri=sample["__key__"],
            source="webdataset"
        )
        
        text = Text.create(
            sample["text"].decode("utf-8"),
            uri=sample["__key__"] + ".txt",
            source="webdataset"
        )
        
        yield PackSequence.from_text_to_image(
            caption=text,
            image=image,
            source="webdataset",
            uri=sample["__key__"]
        )
```

## 常见问题

### Q: 如何处理重复数据？

A: Vault 使用基于内容的哈希 ID，相同内容的数据会自动去重：

```python
# 相同内容的图像会生成相同的 ID
image1 = Image.create(image_bytes, uri="path1", source="src1")
image2 = Image.create(image_bytes, uri="path2", source="src2")
assert image1.id == image2.id  # True
```

### Q: 如何迁移现有数据？

A: 创建自定义 Ingestor 来处理现有数据格式：

```python
class MigrationIngestor(BaseIngestor):
    def process_tasks(self, tasks_chunk, worker_id):
        for old_data in tasks_chunk:
            # 转换旧格式到新格式
            new_sequence = convert_to_pack_sequence(old_data)
            yield new_sequence
```

### Q: 如何备份和恢复数据？

A: 直接复制整个 vault 目录即可：

```bash
# 备份
cp -r /path/to/vault /path/to/backup

# 恢复
cp -r /path/to/backup /path/to/vault
```

### Q: 如何优化查询性能？

A: 使用索引和适当的查询策略：

```python
# 使用索引字段查询
results = storager.meta_hanlder.query_batch(
    "SELECT * FROM sequences WHERE source = ? AND uri LIKE ?",
    ["my_dataset", "/train/%"]
)

# 避免全表扫描
results = storager.meta_hanlder.query_batch(
    "SELECT * FROM sequences WHERE id = ?",
    [sequence_id.to_uuid()]
)
```

### Q: 如何处理大文件？

A: 使用流式处理和分块：

```python
def process_large_file(file_path):
    with open(file_path, 'rb') as f:
        while True:
            chunk = f.read(1024 * 1024)  # 1MB chunks
            if not chunk:
                break
            # 处理 chunk
            yield process_chunk(chunk)
```
